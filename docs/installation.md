# Installation

girelay is currently installable from a source checkout. Public package channels are planned for the first tagged release.

Prerequisites: Git 2.31 or newer and a platform supported by the release
artifacts. Building from source requires Rust 1.85 or newer.

## From a Local Checkout

```bash
cd girelay
cargo install --path crates/girelay
girelay --version
```

Equivalent helper:

```bash
bash scripts/install.sh
```

## Development Build

```bash
cargo build
PATH="$PWD/target/debug:$PATH" girelay --help
```

## From crates.io

After the first public crates.io release:

```bash
cargo install girelay
```

Until then, use the local-checkout installation above.

## From GitHub Releases

After the first public release, download the archive for your platform, verify its checksum, and place `girelay` on your `PATH`.

Planned artifact names:

- `girelay-linux-x86_64.tar.gz`
- `girelay-linux-aarch64.tar.gz`
- `girelay-macos-x86_64.tar.gz`
- `girelay-macos-aarch64.tar.gz`
- `girelay-windows-x86_64.zip`

Each archive contains `girelay` plus an adjacent checksum file.

## From Homebrew

The repository includes a formula template for release automation, but no tap
is advertised until its install command has been tested against a tagged
release.

## Debian And Ubuntu

Checksummed `amd64` and `arm64` `.deb` packages are generated by the Linux release jobs. See [Debian and Ubuntu packages](debian.md).
